# Syntax

### Project setup
```
npm install
```
### Compiles and hot-reloads for development
```
npm run serve
```
### Compiles and minifies for production
```
npm run build
npm install -g serve
serve -s dist
```
### Lints and fixes files
```
npm run lint
```
